# GpsCrystal

Manipuleation des positions GPS (latitude et longitude) - NextJs - MUI

Bonjour Romain,

Comme convenu, tu trouveras ci-dessous le test technique.

Contexte

Le but est de pouvoir manipuler des positions GPS (latitude et longitude).

Définition

Point de vue métier, une position GPS est composée des informations suivantes :
      - Nom
      - Latitude
      - Longitude

A réaliser

Il faut créer une interface qui devra :
      - permettre la saisie de position GPS sous forme d'un wizard (formulaire à étape) :
            - Etape 1 : Saisie du nom
            - Etape 2 : Saisie des coordonnées
            - Etape 3 : Récapitulatif avant sauvegarde
      - permettre de lister les positions GPS enregistrées
      - permettre de mettre à jour une position GPS dans une popup
      - permettre de supprimer une position GPS
      - permettre de donner la distance entre deux positions GPS

Remarques

Le frontend DOIT être sous NextJS + MUI.

Le code DOIT être disponible sur un repository GIT public (github, gitlab, bitbucket etc.)

Tu peux utiliser n'importe quelle autre librairie JS.

La création d'API n'est pas demandée dans le cadre de ce test mais il DOIT y avoir une structuration du code qui permets de comprendre comment cela serait mis en place.

L'interface DOIT être particulièrement travaillée. L'expérience utilisateur DOIT être pris en compte.

Le rendu DOIT être de qualité professionnelle.

Pour rappel, le but pour nous est d'avoir une base de discussion pour l'entretien technique qui suivra.

Si tu as des questions, n'hésite pas à les poser.

Bon courage.
